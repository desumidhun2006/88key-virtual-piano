# 🎹 Virtual Piano

A browser-based **Virtual Piano** that transforms your computer keyboard into a musical instrument.  
This project maps keyboard inputs to **all 88 piano keys** and uses **Tone.js** for real-time sound synthesis, providing a smooth and low-latency musical experience directly in the browser.

---

## 🚀 Features

- 🎹 Full **88-key piano simulation**
- ⌨️ Play notes using your **computer keyboard**
- ⚡ **Low-latency audio synthesis** with Tone.js
- 🌐 Runs **entirely in the browser**
- 🎵 Real-time note playback
- 🧩 Lightweight and beginner-friendly front-end project

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Tone.js**

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/virtual-piano.git
```

2. Open the project folder

```
cd virtual-piano
```

3. Open `index.html` in your browser.
